# Фінальний проєкт з алгоритмів та структур даних

## Опис проєкту

У цьому репозиторії представлено фінальний проєкт з курсу алгоритмів та структур даних.
Проєкт охоплює реалізацію основних алгоритмічних підходів, структур даних, рекурсії,
жадібних алгоритмів, динамічного програмування та методу Монте-Карло з відповідною
візуалізацією результатів.

Кожне завдання реалізоване у вигляді окремого Python-модуля та відповідає критеріям
прийняття фінального проєкту.


## Завдання 7. Метод Монте-Карло

Виконано симуляцію кидання двох гральних кубиків за допомогою методу Монте-Карло
з кількістю симуляцій N = 2000.

### Висновки

Емпіричні ймовірності, отримані методом Монте-Карло, загалом відповідають
аналітичному розподілу сум при киданні двох кубиків.
Найбільшу ймовірність має сума 7, а найменшу — суми 2 та 12.

Невеликі відхилення від аналітичних значень пояснюються випадковим характером
процесу та обмеженою кількістю симуляцій.
Зі збільшенням кількості кидків результати Монте-Карло наближаються до
аналітичних ймовірностей.
